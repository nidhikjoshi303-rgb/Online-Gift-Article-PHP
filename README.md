# Online-Gift-Article-PHP
Online Gift Shop Article is a PHP-based web application for managing and displaying gift products. It includes frontend and admin panel features, allowing users to browse items and admins to manage products with database integration for efficient operations.
How to Run Project (XAMPP)

1. Open XAMPP
2. Start Apache and MySQL
3. Copy project to htdocs folder
4. Open http://localhost/phpmyadmin
5. Create database: giftshop
6. Import database.sql
7. Run:
   http://localhost/giftshop
   http://localhost/giftshop/admin
